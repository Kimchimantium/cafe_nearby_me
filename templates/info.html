Project: Café Finder

This webpage is part of my personal portfolio, showcasing my abilities in web development, particularly in creating a responsive and interactive web application for finding and managing favorite cafés.

Project Overview

The Café Finder is a web application designed to help users discover cafés based on their preferences such as location, seating availability, coffee price, and ratings. It allows users to save their favorite cafés, edit details, and delete them as needed. This project highlights my skills in frontend and backend development, including database management, user interface design, and implementing interactive features like maps and carousels.

Key Features

  • Dynamic listing of cafés with pagination.
  • Interactive Google Maps integration to display café locations.
  • Carousel feature to browse through favorite cafés.
  • Ability to add, edit, and remove cafés from the favorites list.
  • Responsive design for optimal viewing on various devices and screen sizes.

Technologies Used

  • HTML5, CSS3 (Bootstrap for responsive design)
  • JavaScript (for interactive elements)
  • Python Flask (backend framework)
  • SQLAlchemy (for database management)
  • Google Maps API (for map integration)

Challenges and Learning

Throughout the development of the Café Finder, I encountered and overcame several challenges, such as efficiently managing database queries, ensuring the website's responsiveness across different devices, and integrating third-party APIs like Google Maps. This project significantly improved my problem-solving skills, my understanding of web development best practices, and my ability to create a comprehensive user experience.

Future Enhancements

I plan to continue evolving the Café Finder by adding more features, such as user authentication, reviews and comments for each café, and advanced search filters. These enhancements will aim to improve user engagement and provide a more personalized experience.
